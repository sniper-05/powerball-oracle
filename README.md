# Powerball Oracle — Statistical Analysis Engine

An interactive data analytics dashboard analyzing 5 years of 
Powerball draw history (~470 draws, 2021–2026).

## Features
- Frequency analysis for all 69 white balls + 26 Powerballs
- Interactive heatmap visualization
- Pattern detection: consecutive pairs, even/odd splits, sum ranges
- 8-strategy number generator (frequency-weighted, recency-scored, balanced)
- Overdue number tracker

## Tech
Vanilla HTML, CSS, JavaScript — zero dependencies, fully self-contained.

## Key Insight
No statistical method can predict lottery outcomes — 
each draw is independent. This project demonstrates the difference 
between descriptive statistics and predictive validity.
